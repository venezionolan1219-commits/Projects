# Cybersecurity & Networking Labs

This repository contains a mix of Linux‑based cybersecurity labs and a Python socket programming project. It demonstrates practical skills in ethical hacking, network scanning, packet analysis, and client–server communication.

## Contents
- **Ethical Hacking.docx** – Linux ethical hacking tasks, reconnaissance, enumeration, and basic exploitation.
- **NMAP Analysis.docx** – Scan results, open ports, service discovery, and firewall behavior.
- **Wireshark Analysis.docx** – Packet captures, protocol breakdowns, and traffic analysis.
- **server.py** – Python socket server handling connections and basic authentication.
- **client.py** – Python client for sending/receiving messages.
- **users.txt** – Simple credential file used by the server.
- **military_jobs.txt** – Data for server.
